Документация по командам beamchat.

Каждая команда начинается на "/", многие из них имеют аргументы.



Вход в чат:

/joinКоманда работает на @beamju вместо @wowsebot. Команда добавит вас в чат, если вы не в бане.
Не принимает каких-либо аргументов. Добавляет только того, кто ввёл команду.

Команды для всех пользователей:

Алиасы - псевдонимы команд которые можно использовать вместо основных.

/color кодцветаКод цвета это значение цвета, которое поддерживает html. Например hex и rgb. Отправляет вам картинку с цветом и оттенками

/notify цифратип текстКоманда выводит определенный текст через определенное время. Цифра - количество заданного типа. Тип - тип едениц - минуты/секунды/часы - m/s/h. Текст - текст, который будет отправлен. Пример команды: /notify 8m хуй. Максимальное время: 8 часов.

/encrypt текстЗашифровывает текст по AES256 со случайным ключем от 1 до 26 символов и выводит зашифрованные данные и ключ.

/encryptkey ключ текстЗашифровывает текст по AES256 с ключем, заданным пользователем, и выводит зашифрованные данные и ключ.

/decrypt ключ текстРасшифровывает информацию по ключу и данным и выводит результат.

/passgen длинна составГенерирует пароль заданной длинны, состоящий из определенных символов. Если не указывать состав то будут исползоваться символы a-z A-Z 0-9. Также текст в скобках из состава берется за часть. Т.е. 123(kek) будет добавлять kek вместе.

/qr текстКоманда отправляет картинку с qr кодом содержащим текст, который пользователь отправил.
Также можно использовать reply на другом сообщении без указывания аргумента. Тогда возьмется текст сообщения на которое вы сделали reply.
Алиасы: /qrcode

/tts текстКоманда выводит войс с текстом, который пользователь отправил.
Также можно использовать reply на другом сообщении без указывания аргумента. Тогда возьмется текст сообщения на которое вы сделали reply. Максимум 180 символов.
Алиасы: /voice, /speech

/bash номерКоманда выводит цитату с bash.im по номеру.
Если номер не указан то выводится случайная цитата.

/nowКоманда выводит текущие дату и время.
Алиасы: /time, /date

/unixtimeКоманда выводит текущее время одним числом.

/latex кодОтправляет картинки со сгенерированным latex кодом
Алиасы: /tex

/weather местоОтправляет температуру, данные по облачности и скорость и направление ветра в месте. Местом может быть город или zip-код.

/fortuneОтправляет вывод программы fortune.

/anime названиеИщет и отправляет информация о аниме.

/rollВыдает случайный стикер с числом
/roll числоВыдает случайное число до введенного
/roll число-числоВыдает случайное число между двумя числами
/roll числоdчислоВыдает номер чисел (число до d) до второго числа (число после d)
Алиасы: /dice, /random

/rev текстВыведет текст наоборот. Вместо текста можно использовать reply.
Алиасы: /reverse

/chcase текстВыведет текст в другой раскладке. Т.е. kjk переведет в лол, руддщ переведет в hello. Также работает на reply.
Алиасы: /changecase

/usersОтправит список пользователей и их id.
Алиасы: /userlist, /chatids

/translit текстпереводит текст в транслит и из транслита.

/hb текстЗагрузит текст на hastebin.com и даст ссылку. Вместо текста можно использовать reply.
Алиасы: /haste, /hastebin

/upper текстОтправляет текст КАПСОМ
Алиасы: /uppercase

/invert текстОтправляет текст, где все буквы написаны обратным размером.
Алиасы: /invertcase

/lower текстОтправляет текст маленькими буквами
Алиасы: /lowercase

/capitalize текстОтправляет текст Где Каждое Слово Написано С Большой Буквы

/tolisp текстОтправляет текст, (который (выглядит (как (псевдокод (языка (программирования (lisp)))))))

/htr #КОДКонвертирует hex цвет в rgb цвет. Поддерживается как короткий, так и длинный вариант.
Алиасы: /hextorgb, /hex2rgb

/lorem количествоВыдает определенное количество слов текста lorem ipsum. Максимум: 600
Алиасы: /lipsum, /loremipsum

/pi длиннаВыдает число пи определенной длинны. Максимум: 1600
Алиасы: /π

/uuidВыдает случайный UUID/GUID
Алиасы: /guid

/gist расширение текстОтправляет на gist.github.com текст под определенным расширением и шлет ссылку.

/unsplash random/latest/searchОтправляет картинки с unsplash.com. Если выбрать random то отошлется случайная картинка, если latest то последняя, загруженная на сайт, если search то после search надо указать запрос, тогда будет произведен поиск.

/rth rgb(красный,зеленый,синий)Конвертирует rgb цвет в hex цвет.
Алиасы: /rgbtohex, /rgb2hex

/googl ссылкаСократит ссылку через goo.gl и отправит короткий вариант.

/bitly ссылкаСократит ссылку через bit.ly и отправит короткий вариант.

/wm текстВаше сообщение будет перенаправлено @fedora
Алиасы: /warnmemju

/wiki текстНайдет информацию по запросу и отошлет вам.
Алиасы: /wikipedia, /pediwikia

/tr текстОпределит язык и переведет текст на Русский.
/tr язык текстПереведет текст с выбранного языка на Русский.
/tr язык1 язык2 текстПереведет текст с языка 1 на язык 2.
Обозначение языка должно состоять из первых двух букв названия языка.
Например: японский будет ja, русский - ru.
Также принимает reply.
Алиасы: /translate

/lol количествоГенерирует смех из символов азх определенной длинны. Максимум 1000.
/lolГенерирует смех из символов азх и отправляет 1 раз

/nyashБерет случайную картинку из nya.sh

/infoОтправляет информацию о вас. Если есть reply, то того, на кого reply.

/countШлет 3, 2, 1. Иногда удобно.
Алиасы: /timer

/wscr ссылкаОтсылает документом скриншот веб страницы шириной 1000px
Алиасы: /webshot

/html кодОтсылает документом скриншот выполненного html кода шириной 1000px
Алиасы: /hscr

/md кодКомпилирует markdown код, отправляет html код и картинку html кода.
Алиасы: /markdown

/js кодВыполняет javascript код с направлением результата в чят. Дан минимум команд, Таймаут: 5s. Вывод (console.log) один на весь код.
Алиасы: /nodejs, /javascript, /node

/range от доЕсли от и до - цифры, то отправляются все числа между заданными включая заданные. Если другие символы, то все символы между двумя символами включая заданные. Если результат очень большой и не вмещается в одно сообщение то он отправляется в личные сообщения тому, кто запросил, несколькими сообщениями.
Алиасы: /between

/img кодИщет в google картинки и отправляет.
Алиасы: /image

/g кодИщет в google и отправляет несколько результатов.
Алиасы: /google, /search, /srch

/piratebay текстИщет в piratebay торренты и отправляет ссылки.
Алиасы: /torrent

/imdb названиеИщет в imdb.com.

/zalgo текстОтправляет т͐͂е̋ͥк ͛͊сͯ̇тͧ̄ в z̓̆a͌ͮlͫ ̇̋gͯ̍ōͩ.

/banish текствzalgoКонвертирует zalgo-текст в нормальный.
Алиасы: /dezalgo, /unzalgo

/qОтправляет случайную цитату. С reply отсылает цитату пользователя на которого был сделан reply.
/q номерОтправляет цитату по номеру.
/q номер+Отправляет рандомную цитату из списка с айди больше номера.
/q текстВыдает случайную цитату содержащую текст.
/q latestВыдает последнюю цитату.
/q statsВыдает топ людей в цитатнике
Алиасы: /quote

/leaveКикает вас из конфы. Вы сможете вернуться по ссылке.
Алиасы: /cъебываю

/banlistПоказывает список всех пользователей, которых забанили
Алиасы: /пидары, /eblanlist

/modlistПоказывает список всех модераторов
Алиасы: /moderlist, /moderatorlist

/aboutПоказывает описание beamchat

Команды администраторов и модераторов:

/kick имяКикает пользователя по @username (@ не обязательно). Если его нет, то можно кикать по reply. Он сможет зайти.
Алиасы: /убери, /кик
/kickidКикает пользователя по ID. Он сможет зайти.
Алиасы: /убериайди, /кикайди

/ban имяЗабанит пользователя по @username (@ не обязательно), если оно есть. Иначе можно банить пользователя по reply. Он больше не зайдет.
Алиасы: /бань, /бан
/banid айдиБанит пользователя по ID. Он больше не зайдет.
Алиасы: /баньайди, /банайди

/unban имяРазбанит пользователя по @username (@ не обязательно), если оно есть. Иначе можно разбанить пользователя по reply. Он теперь опять сможет заходить.
Алиасы: /разбань, /разбан
/unbanidРазбанит пользователя по ID. Он теперь опять сможет заходить.
Алиасы: /разбаньайди, /разбанайди

Команды только администраторов:

/mod имяДаст права модератора пользователю по @username (@ не обязательно). Если его нет, то по reply. Он сможет выполнять команды выше.
/modidДаст права модератора пользователю по ID. Он сможет выполнять команды выше.

/unmodУберёт права модератора у пользователя по reply. Он не сможет выполнять команды выше.
/unmodidУберёт права модератора у пользователя по ID. Он не сможет выполнять команды выше.

/warn текстОтошлет текст трижды в beamchat.

/bc текстОтошлет текст в beamchat. Можно отсылать как @wowsebot так и @beamju (и в лс).

/scrОтправляет скриншот компьютера администратора.

/saveСохранить цитату по reply
Алиасы: /q save, /quote save
/q remove номерУдалить цитату по номеру.
Алиасы: /quote remove

/kb массивыПоказывает клавиатуру по массивам
/kbsm массивыПоказывает маленькую клавиатуру по массивам
/hidekbУбирает клавиатуру

/bot stopЗавершает процесс бота. Его можно запустить только через компьютер, на котором запущен бот.

/bot offВключает игнорирование всех команд кроме /bot on.

/bot onВыключает игнорирование всех команд если оно включено (через /bot off).

/shell кодИсполняет bash код.

/clearЗаполняет чат большим количеством невидимых сообщений.
